# Object-Oriented Programming Project 2025.1 - Group 12

## Description

This project was developed as part of the **Object-Oriented Programming (OOP)** course in the 2025.1 semester and was conducted under the guidance of **Dr. Nguyễn Thị Thu Trang**.

The project is primarily implemented in **Java**, focusing on designing a flexible, maintainable, and scalable system. It demonstrates how to organize code according to OOP principles while providing students with hands-on experience in implementing structured and efficient programming techniques.
